# ShakeDrop — 设计规范

## 设计原则

- **极简**：只展示必要信息，无冗余装饰
- **毛玻璃**：使用 macOS 原生材质，与系统风格统一
- **即时反馈**：每个交互都有明确视觉响应

## 色彩

| 用途 | 浅色模式 | 深色模式 |
|------|---------|---------|
| 面板背景 | NSVisualEffectView .hudWindow | 同左（系统自适应） |
| 拖放区域边框（默认） | 白色半透明，线宽 2pt，虚线 | 同左 |
| 拖放区域边框（悬停） | 系统蓝色 #007AFF，发光 shadow | 同左 |
| 文字主色 | .primary | .primary |
| 文字辅助色 | .secondary | .secondary |
| 删除按钮 | 红色 #FF3B30 | 同左 |
| 文件图标 | 系统文件图标（NSWorkspace icon） | 同左 |

## 面板布局（380 x 280 pt，圆角 16pt）

```
┌──────────────────────────────────┐  ← 毛玻璃背景
│  ┌────────────────────────────┐  │
│  │      拖放区域               │  │  ← 空状态：180pt 高
│  │   (虚线边框 Drop Zone)      │  │      有文件：60pt 高
│  │       ↓                    │  │
│  │   拖拽文件到此处暂存         │  │
│  └────────────────────────────┘  │
│                                  │
│  ┌────────────────────────────┐  │
│  │ 📄 项目文档.pdf       ✕    │  │  ← 文件列表
│  │ 📁 图片素材/          ✕    │  │     滚动区域
│  │ 🖼 截图.png           ✕    │  │
│  └────────────────────────────┘  │
│                                  │
│  共 3 个文件         [全部清除]   │  ← 底部状态栏
└──────────────────────────────────┘
```

## 动画规范

| 场景 | 动画 | 参数 |
|------|------|------|
| 面板弹出 | 缩放 + 淡入 | scale 0.8→1.0, opacity 0→1, 0.25s |
| 面板关闭 | 缩放 + 淡出 | scale 1.0→0.8, opacity 1→0, 0.15s |
| 拖入悬停 | 边框颜色过渡 + shadow 增强 | 0.2s ease-out |
| 文件插入 | 弹性滑入 | spring: response 0.4, damping 0.6 |
| 文件删除 | 右滑淡出 + 列表收起 | 0.25s ease-in |
| 拖出文件 | 列表项高亮 + 缩放 | 0.15s |
| 全部清除 | 列表整体淡出 | 0.3s ease-out |

## 字体

| 用途 | 字体 | 大小 |
|------|------|------|
| Drop Zone 提示文字 | SF Pro Text | 14pt, .secondary |
| 文件名 | SF Pro Text | 13pt, .primary |
| 底部状态 | SF Pro Text | 12pt, .secondary |
| 按钮 | SF Pro Text | 12pt, .primary |

## 间距

- 面板内边距：16pt
- 文件行间距：8pt
- 文件行内边距：8pt (水平), 6pt (垂直)
- 图标尺寸：24x24pt
- 删除按钮尺寸：16x16pt

## 图标

- 状态栏：SF Symbol `tray.and.arrow.down`，16pt
- 空状态提示箭头：SF Symbol `arrow.down.to.line`
- 删除：SF Symbol `xmark.circle.fill`，12pt
