# ShakeDrop — 需求规格

## 概述

ShakeDrop 是一个 macOS 菜单栏工具，让用户在 Finder 中拖拽文件时通过「甩动鼠标」触发一个文件暂存面板，将多个文件收集后一次性移动到目标位置。

## 用户故事

1. **快速收集文件**：用户在 Finder 中按住文件拖拽，快速甩动鼠标 1.5 秒，弹出一个暂存面板，将当前拖拽的文件放入其中。
2. **批量暂存**：用户可以继续从不同位置拖入更多文件到面板。
3. **一次性取出**：用户可以从面板中一次性拖出所有暂存文件到目标 Finder 目录，文件被移动（剪切）到目标。
4. **保持打开**：文件取出后面板不关闭，可继续暂存新文件。

## 功能需求

| 编号 | 功能 | 描述 |
|------|------|------|
| F1 | 甩动触发 | 在 Finder 按住文件拖拽时，甩动鼠标 1.5 秒触发面板弹出 |
| F2 | 面板弹出 | 面板以毛玻璃效果出现在鼠标光标附近 |
| F3 | 文件拖入 | 支持拖入文件和文件夹到面板 |
| F4 | 文件去重 | 相同路径的文件不重复添加 |
| F5 | 文件列表 | 以图标+文件名列表形式展示暂存文件 |
| F6 | 拖出移动 | 支持一次性拖出所有文件，执行移动（剪切）操作 |
| F7 | 单文件删除 | 支持从列表中移除单个文件 |
| F8 | 全部清除 | 一键清空所有暂存文件 |
| F9 | 面板保持 | 文件拖出后面板保持打开 |
| F10 | 手动关闭 | 支持 ESC 或点击面板外部关闭面板 |
| F11 | 菜单栏 | 状态栏图标 + 下拉菜单（显示/隐藏面板、退出） |
| F12 | 权限引导 | 检测辅助功能权限，无权限时引导开启 |
| F13 | 冷却机制 | 触发后 2 秒内不重复弹出 |

## 非功能需求

| 编号 | 需求 | 描述 |
|------|------|------|
| NF1 | 性能 | 甩动检测延迟 < 200ms |
| NF2 | 稳定性 | 文件移动失败时提供明确错误提示，不崩溃 |
| NF3 | 兼容性 | 支持 macOS 14.0+，原生 ARM 架构 |
| NF4 | 极简设计 | 毛玻璃背景、圆角、无冗余 UI 元素 |

## 设计约束

- 使用 SwiftUI + AppKit 原生开发
- 菜单栏应用，LSUIElement = YES（不显示 Dock 图标）
- 需要辅助功能权限（Accessibility）
